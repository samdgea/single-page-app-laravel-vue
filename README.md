See application on action by clicking [this link](https://abdilah/s/10006) or see the blog post by clicking [here](https://abdilah/s/10005).

## Inisialisasi

Jalankan perintah dibawah ini secara berurutan.

```
$ composer install
$ npm install
$ cp .env.example .env
$ php artisan key:generate
$ php artisan serve
$ npm run watch
```
